This app is a real-time poker chip tracker that can be used for any poker game when chips aren't available, or you don't have a poker set, only a deck of cards.

Made with:
Flask-SocketIO (Backend)
React + Vite + Tailwind CSS (Frontend)

Features:
Room security with 5-character codes, 
Real-time player updates via Socket.IO, 
Track chip counts for each player, 
Configurable starting and blind amounts, 
Up to 10 players per room, 
Action log, 
Room Lead (No Cheating!)


How to Use (For Now)
1. Clone this repo
2. Set up a Python virtual environment
3. Install Python dependencies
4. Install frontend dependencies
5. Configure urls
6. Run python app.py in terminal to start game engine
7. Run cd frontend && npm run dev to get a visual
8. Go tohttp://localhost:5173 (or 3000 depending on your setup)
9. Enjoy!
10. You can also wait till I deploy this but up to you!
